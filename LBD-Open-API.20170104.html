    猎必得 Open API 平台开发概述

    猎必得（以下简称：LBD） Open API 平台主要是提供给第三方合作伙伴分享其招聘数据给猎必得平台使用的一个接口平台。

    具体业务时序如下：

    接口返回码说明 ¶

    每次调用平台接口时，可能获得正确或错误的返回码，开发者可以根据返回码信息调试接口，排查错误。

    全局返回码说明如下：

    返回码 说明
    100 不合法的{字段名称}数据
    101 请阅读并同意《猎必得用户协议》
    102 当前用户在猎必得平台中已加入另一家公司
    103 当前公司、用户信息已经存在猎必得平台，无需重复授权。
    104 提交的公司Id不存在
    105 提交的用户Id不存在
    106 提交的职位Id不存在
    107 请求的文件Id不存在
    108 提交的候选人Id不存在
    109 第三方平台处理失败
